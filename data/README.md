# Data

This folder contains the raw dataset used in the project.

## Included File

### `plants.data`

This file contains plant names and the geographic regions where each plant occurs. Each row is comma-separated:

```text
plant name,region_1,region_2,...
```

The first value is the plant name. The remaining values are region abbreviations.

## Source and License

The data used for this project is the public Plants dataset from the UCI Machine Learning Repository. The original project notebook notes that the dataset is available under the Creative Commons Attribution 4.0 International license.

When reusing the dataset, cite the original UCI Machine Learning Repository source and follow the dataset license terms.

## Processed Outputs

Processed tables and analysis outputs are written to the `reports/` folder rather than this folder.
